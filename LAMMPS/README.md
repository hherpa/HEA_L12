# LAMMPS — Молекулярная динамика HEA Al-Co-Cr-Fe-Ni-Ti

Моделирование высокоэнтропийного сплава **Al-Co-Cr-Fe-Ni-Ti** методом молекулярной динамики в пакете **LAMMPS**.

Потенциал: **MEAM** (Modified Embedded Atom Method)
Единицы: **metal** (Å, эВ, пс, К, бар)

---

## Система

### Состав сплава

| # | Тип LAMMPS | Элемент |
|---|------------|---------|
| 1 | type 1 | Fe |
| 2 | type 2 | Ni |
| 3 | type 3 | Ti |
| 4 | type 4 | Cr |
| 5 | type 5 | Co |
| 6 | type 6 | Al |

Эквиатомный состав — каждый элемент занимает **~16.67%** сайтов.

### Начальная структура

- **Решётка:** BCC
- **Постоянная решётки:** a = 2.87 Å
- **Суперячейка:** 10×10×10
- **Всего атомов:** 2000
- **Границы:** периодические (p p p)

### Присваивание типов

```lammps
set atom * type/ratio 2 0.16666667 12345
set atom * type/ratio 3 0.16666667 12346
set atom * type/ratio 4 0.16666667 12347
set atom * type/ratio 5 0.16666667 12348
set atom * type/ratio 6 0.16666667 12349
```

Случайное распределение с фиксированными seed (12345–12349) для воспроизводимости.

## Потенциал MEAM

### Загрузка

```lammps
pair_style      meam
pair_coeff      * * library.meam Fe Mn Ni Ti Cu Cr Co Al FeMnNiTiCuCrCoAl.meam Fe Ni Ti Cr Co Al
```

### `library.meam`

Одноэлементные параметры для 8 элементов:

| Элемент | Структура |
|---------|-----------|
| Al | fcc |
| Co | hcp |
| Cu | fcc |
| Cr | bcc |
| Fe | bcc |
| Mn | hcp |
| Ni | fcc |
| Ti | hcp |

Каждый элемент: когезионная энергия, равновесное расстояние, модуль объёмного сжатия, параметры экранирования.

### `FeMnNiTiCuCrCoAl.meam`

Параметры бинарных и тройных взаимодействий:
- **Cmin / Cmax** — параметры экранирования для тройных взаимодействий
- **lattce, Ec, alpha, re** — параметры парного потенциала
- **attrac, repuls, nn2** — множители притяжения/отталкивания
- Покрывает все 28 бинарных пар
- Глобальные настройки: `rc = 5.0`, `delr = 0.1`, `ialloy = 2`

### Источник и цитирование

Потенциал взят из NIST/CTCMS: [2025--Sharifi-H-Wick-C-D--Fe-Mn-Ni-Ti-Cu-Cr-Co-Al](https://www.ctcms.nist.gov/potentials/entry/2025--Sharifi-H-Wick-C-D--Fe-Mn-Ni-Ti-Cu-Cr-Co-Al/)

**Citation:** H. Sharifi, and C.D. Wick (2025), "Developing interatomic potentials for complex concentrated alloys of Cu, Ti, Ni, Cr, Co, Al, Fe, and Mn", *Computational Materials Science* 248, 113595. DOI: [10.1016/j.commatsci.2024.113595](https://doi.org/10.1016/j.commatsci.2024.113595).

### Почему 6 из 8 элементов

Оригинальный потенциал охватывает 8 элементов: **Cu, Ti, Ni, Cr, Co, Al, Fe, Mn**. Наш сплав **Al-Co-Cr-Fe-Ni-Ti** использует 6 из них. Элементы **Cu** и **Mn** присутствуют в файлах потенциала (для полноты бинарных/троичных комбинаций), но не назначаются атомам в нашей системе.

### Назначение и ограничения потенциала

**Для чего оптимизирован:**
- Структурные свойства HEA и сложных концентрированных сплавов (CCA)
- Прочность на сдвиг и постоянные упругости
- Динамика дислокаций и их влияние на прочность
- Дефектные эффекты (пустоты и их влияние на свойства материала)

**Не оптимизирован для:**
- Температурно-зависимых свойств
- Плотности
- Коэффициентов теплового расширения
- Данных теплопроводности

> **Важно:** Результаты этапов, связанных с нагревом/охлаждением (шаги 2–4), следует интерпретировать с учётом этих ограничений. Потенциал корректно описывает структурную релаксацию и механический отклик, но не предназначен для точного моделирования термических свойств.

## Алгоритм симуляции

### Минимизация энергии

Перед началом динамики — минимизация начальной конфигурации:
```lammps
minimize  1.0e-6  1.0e-8  100  1000
```

### Шаг 1: Релаксация объёма при 300K

```lammps
fix  2 all npt temp 300 300 0.1 iso 0 0 0.1
run  100
```

NPT-ансамбль. Термостат 300K, баростат 0 бар, изотропное сжатие/расширение. Снятие внутренних напряжений начальной структуры.

**Результат (тестовый):** объём 23640 → 24400 Å³, давление стабилизировалось.

---

### Шаг 2: Высокотемпературный отжиг (гомогенизация) при 1500K

```lammps
fix  2 all npt temp 1500 1500 0.1 iso 0 0 0.1
run  100
```

1500K — активная диффузия, гомогенизация распределения элементов. NPT позволяет тепловое расширение.

**Результат (тестовый):** объём вырос с 24436 до 26106 Å³.

---

### Шаг 3: Охлаждение 1500K → 300K

```lammps
fix  3 all npt temp 1500 300 0.001 iso 0 0 0.1
run  200
```

Линейное охлаждение с шагом dt = 0.001 пс. Контролируемое снижение температуры с сохранением изотропного давления.

---

### Шаг 4: Низкотемпературный отжиг (SRO) при 700K

```lammps
fix  4 all npt temp 700 700 0.1 iso 0 0 0.1
run  100
```

700K — развитие ближнего порядка (Short-Range Order). Атомы занимают энергетически выгодные позиции.

---

### Шаг 5: Финальная релаксация при 300K

```lammps
fix  5 all npt temp 300 300 0.1 iso 0 0 0.1
run  100
```

Закрепление равновесия после отжига. Система готова к механическим испытаниям.

**Результат (тестовый):** температура 300±15K, энергия ~-8543 эВ, объём стабилен.

---

### Шаг 6: Одноосное растяжение

```lammps
fix  6 all nvt temp 300 300 0.1
fix  7 all deform 1 x erate 0.001 remap x
run  100
```

- **Ансамбль:** NVT (термостат 300K)
- **Деформация:** вдоль оси X, скорость 0.001 пс⁻¹
- **Remap:** координаты атомов масштабируются вместе с ячейкой
- **Вывод:** `stress_strain.csv` (Strain, Stress_GPa)

Для полного растяжения до 10% деформации требуется **100 000 шагов** (dt = 0.001 пс, srate = 0.001 пс⁻¹).

## Постобработка

Ноутбук `logs/LAMMPS_post_processing.ipynb` — чтение CSV-логов через pandas, разделение по `;`, базовый анализ.

Формат CSV:
```
Step;Temp;Press;TotEng;Volume
20;0.238;18343.6;-8559.8;23659.8
```

Формат stress_strain.csv:
```
Strain;Stress_GPa
0.00062;0.02
```
