# SQS Generator — Генератор специальных квазислучайных структур

Генерация **Special Quasirandom Structure (SQS)** для упрочняющей фазы **(Ni,Co,Fe,Cr)₃(Al,Ti)** со структурой **L12** (Pm-3m, #221).

Инструмент: библиотека **sqsgenerator** (C++ backend) + **pymatgen** для валидации.

## Состав и подрешётки

Структура L12 имеет две подрешётки:

- **B-сайт (1a)** — углы куба: **Al, Ti** (соотношение 3:1)
- **A-сайты (3c)** — центры граней: **Ni, Co, Fe, Cr** (соотношение 12:5:4:3)

| Размер | Суперячейка | B-сайтов | A-сайтов | Итерации |
|--------|-------------|----------|----------|----------|
| 32 атома | 2×2×2 | 8 (Al:6, Ti:2) | 24 (Ni:12, Co:5, Fe:4, Cr:3) | 10 000 000 |
| 64 атома | 2×2×4 | 16 (Al:12, Ti:4) | 48 (Ni:24, Co:10, Fe:8, Cr:6) | 50 000 000 |
| 128 атомов | 4×4×2 | 32 (Al:24, Ti:8) | 96 (Ni:48, Co:20, Fe:16, Cr:12) | 50 000 000 |

Параметр решётки: **a = 3.57 Å**

## Теория SQS

### Что такое SQS?
**SQS (Special Quasirandom Structure)** — это метод замены бесконечного случайного сплава конечной периодической суперячейкой, которая максимально точно воспроизводит статистические характеристики идеального случайного раствора. 

В реальном случайном сплаве атомы распределены хаотично. В DFT/MD мы не можем смоделировать бесконечный объём, поэтому берём маленькую ячейку и подбираем такое расположение атомов, чтобы **корреляционные функции кластеров** (пары, тройки, четвёрки атомов на определённых расстояниях) совпадали с теоретическими для полностью случайного сплава.

### Что такое `Objective`?
Это **целевая функция оптимизации**, которую минимизирует `sqsgenerator`. Математически это взвешенная сумма квадратов отклонений реальных корреляций от идеальных случайных:

$Objective = Σ wₖₘ · (Πₖₘ − Πₖₘ^random)²$

где:
- $Πₖₘ$ — реальная корреляционная функция для кластера типа `k` (пара, тройка и т.д.) в оболочке `m`
- $Πₖₘ^{random}$ — теоретическое значение для идеального случайного сплава
- $wₖₘ$ — весовой коэффициент (обычно убывает с расстоянием, пары важнее троек, тройки важнее четвёрок)

**Чем ближе Objective к 0, тем точнее ячейка имитирует истинную случайность.**

### Выбор лучшего SQS
**Строго по минимуму Objective.** 

В выводе `sqsgenerator` структуры отсортированы по возрастанию этой величины. `Objective #0` — всегда лучший доступный вариант. Если несколько структур имеют почти одинаковый минимальный objective, можно выбрать любую (разница будет в несущественных дальних корреляциях).

## Алгоритм работы ноутбука

1. **Установка зависимостей** — `sqsgenerator`, `pymatgen`
2. **Генерация координат** — построение идеальной L12 решётки с заполнением B-сайтов (Al) и A-сайтов (Ni)
3. **Конфигурация sqsgenerator** — `sublattice_mode: split` для раздельной оптимизации A и B подрешёток
4. **Запуск оптимизации** — перебор конфигураций, минимизация Warren-Cowley параметров SRO
5. **Экспорт лучшего SQS** — сохранение в CIF с очисткой формата (`sed 's/0+//g'`)
6. **Валидация через pymatgen** — проверка формулы, числа атомов, состава по элементам
