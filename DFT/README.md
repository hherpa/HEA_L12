# DFT (density functional theory) - Теория функционала плотности

Метод расчёта электронной структуры систем многих частиц в квантовой физике и квантовой химии. Это один из наиболее широко используемых и универсальных методов в вычислительной физике и вычислительной химии

> **Псевдопотенциал** — термин, который означает эффективный потенциал, используемый в физике для упрощённого описания сложных систем. Например, в теории металлов псевдопотенциал — это потенциал, действующий на электроны проводимости со стороны ионов.

Валентные электроны, отвечающие за химическую связь, не могут быть описаны плоскими волнами вблизи ядра из-за сильных осцилляций их волновых функций. Псевдопотенциал убирает эти осцилляции, создавая гладкие «псевдоволновые функции», которые требуют для разложения намного меньше плоских волн. Это делает расчёты практичными.

## Выбор псевдопотенциалов

> `ecutwfc (Wavefunction energy cutoff)` — это кинетическая энергия отсечения для волновых функций (в единицах Ридберга, Ry). Параметр задаёт верхний предел энергии для базисных функций (плоских волн), используемых для разложения электронных волновых функций в расчётах.

### Типы псевдопотенциалов

1. `NC (Norm-conserving)` — это тип псевдопотенциалов, в котором норма псевдоволновой функции внутри радиуса обрезания совпадает с нормой полной (all-electron) волновой функции. Такие псевдопотенциалы требуют относительно высокого ecutwfc (обычно 60–100 Ry) и совместимы с гибридными функционалами и DFPT.

2. `PAW (Projector Augmented Wave)` — это метод, который восстанавливает полную (all-electron) волновую функцию и плотность внутри атомных сфер, но при этом работает с гладкими псевдо-функциями в межъядерном пространстве. Он точнее norm-conserving, требует меньшего ecutwfc (обычно 25–50 Ry), но сложнее в реализации и немного медленнее на итерацию. PAW — текущий стандарт высокой точности в QE.

3. `US (ultrasoft pseudopotentials)` — ультрамягкие псевдопотенциалы. Это компромисс между NC и PAW. Они позволяют использовать ещё меньшую энергию отсечки (ecutwfc ~20-40 Ry) за счёт того, что внутри core-области псевдоволновая функция может быть "неправильной" по норме, и это компенсируется вспомогательными функциями (augmentation charges). Они менее точны, чем PAW, но быстрее. В QE широко используются, например, наборы GBRV. Но для некоторых задач (гибридные функционалы, DFPT) могут быть проблемы.

Выбор типа псевдопотенциалов нужно делать с учетом выбора функционала. Выбор функционала определяется тремя факторами:
1. *Типом вашей системы*
2. *Целевыми свойствами*
3. *Доступными ресурсами*

### Приближения в теории функционала плотности (DFT) и их функционалы

1. **LDA (Local Density Approximation — приближение локальной плотности)**

    1. **VWN (Vosko-Wilk-Nusair)** — параметризация корреляционной энергии однородного электронного газа на основе данных Монте-Карло.
    2. **PZ (Perdew-Zunger)** — параметризация, основанная на данных Синдера, Ландека и др. (часто используется как реализация LDA).
    3. **PW92 (Perdew-Wang 1992)** — высокоточная параметризация корреляционной энергии LDA от Пердью и Вана.
    4. **HL (Hedin-Lundqvist)** — одна из ранних параметризаций LDA, основанная на теории Хэдина — Лундквиста.

2. **GGA (Generalized Gradient Approximation — приближение обобщённого градиента)**

    1. **PBE (Perdew-Burke-Ernzerhof)** — наиболее популярный GGA-функционал, не содержащий подгоночных параметров (кроме фундаментальных констант).
    2. **PBEsol (PBE revised for solids)** — модификация PBE, оптимизированная для описания свойств твёрдых тел и поверхностей.
    3. **BLYP (Becke-Lee-Yang-Parr)** — комбинация обменной части Бике (B88) и корреляционной Ли, Янга, Парра (LYP).

3. **meta-GGA (meta-Generalized Gradient Approximation — мета-приближение обобщённого градиента)**

    1. **SCAN (Strongly Constrained and Appropriately Normed)** — функционал, удовлетворяющий всем известным точным ограничениям для мета-GGA.
    2. **TPSS (Tao-Perdew-Staroverov-Scuseria)** — первый неэмпирический мета-GGA-функционал.
    3. **M06-L (Minnesota 2006, local)** — локальный мета-GGA-функционал из семейства Миннесотских, включает эмпирическую подгонку.

4. **Hybrid functionals (Гибридные функционалы)**

    1. **B3LYP (Becke, 3-parameter, Lee-Yang-Parr)** — один из самых известных гибридных функционалов, смешивает 20% точного обмена Хартри — Фока.
    2. **PBE0 (PBE with 0% screening, иногда PBE1PBE)** — гибридная версия PBE с 25% точного обмена.
    3. **HSE06 (Heyd-Scuseria-Ernzerhof 2006)** — range-separated гибридный функционал (короткодействующая часть гибридная, длиннодействующая — PBE).
    4. **M06-2X (Minnesota 2006, 2X)** — высоко-нелокальный гибридный функционал с 54% точного обмена, оптимизирован для невалентных взаимодействий и термохимии.
