# Псевдопотенциалы для HEA L12 (Al-Co-Cr-Fe-Ni-Ti)

## Основные параметры
- **Тип**: PAW (Projector Augmented Wave)
- **Функционал**: PBEsol (GGA)
- **Формат**: UPF v2.0.1 (совместим с Quantum ESPRESSO)

## Источники файлов

### 1. SSSP Library (Materials Cloud)
- **Элементы**: Al, Fe
- **Ссылка**: https://legacy.materialscloud.org/discover/sssp/table/efficiency
- **Библиотека**: PSLibrary

### 2. PseudoDojo
- **Элементы**: Co, Cr, Ni, Ti
- **Ссылка**: https://www.pseudo-dojo.org/
- **Настройки**: Functional=PBEsol, Type=PAW
