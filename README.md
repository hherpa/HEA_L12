# HEA_L12 — Моделирование высокоэнтропийного сплава

Компьютерное исследование упрочняющей фазы **L12** (Ni,Co,Fe,Cr)₃(Al,Ti) в высокоэнтропийном сплаве **Al-Co-Cr-Fe-Ni-Ti**.

> Проект охватывает полный цикл: от генерации атомных структур до расчётов из первых принципов и молекулярной динамики.

---

## Структура проекта

| Директория | Описание |
|------------|----------|
| **`CIF/`** | Генерация **SQS** (Special Quasirandom Structures) структур (32, 64, 128 атомов) с помощью `sqsgenerator` |
| **`DFT/`** | Расчёты электронной структуры и релаксация (**Quantum ESPRESSO**, **GPAW**, **PySCF**), псевдопотенциалы |
| **`LAMMPS/`** | Молекулярная динамика (потенциал **MEAM**): отжиг, охлаждение, одноосное растяжение и постобработка |

---

## Состав сплава

- **Al, Co, Cr, Fe, Ni, Ti**
- Структура: **L12** (Pm-3m, #221)
- Подрешётки:
  - **B-сайт (1a)**: Al, Ti
  - **A-сайт (3c)**: Ni, Co, Fe, Cr
